Web Design

The term "web design" is normally used to describe the design process relating to the front-end (client side) design of a website including writing markup. Web design partially overlaps web engineering in the broader scope of web development. Web designers are expected to have an awareness of usability and be up to date with web accessibility guidelines.

UI/UX Design

It’s true that a UI (user interface) or UX (user experience) designers’ main aim is to enhance users’ satisfaction, by creating products and interfaces that are intuitive, simple and efficient. UI is about the visual design of the product interface. UX is about the user, or rather, human experience. It’s about analyzing people’s intents when they use the product

App Design

App design combines the user interface (UI) and user experience (UX). While UI lends itself to the overall style of the app (including the colors, fonts, and general look and feel), UX focuses on the actual functionality and usability.
